# Teslo API

1. Clonar proyecto
2. Instalar los paquetes con yarn:
```
yarn install
```
3. Clonar el archivo ```.env.template ``` y renombrarlo a ``` .env```
4. Cambiar las variables de entorno en ```.env```
5. Levantar la base de datos
```
docker-compose up -d
```

